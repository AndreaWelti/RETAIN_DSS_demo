Chi usa il tool e come?

Il profilo utente determina il formato dell'interfaccia

A

Ricercatori / Data scientists

Usano Jupyter Notebook o script Python. Vogliono controllare il codice, esplorare i risultati, modificare parametri. Interfaccia: notebook o CLI.

B

Ingegneri di processo / operatori

Non vogliono vedere codice. Inseriscono le proprietà del materiale in ingresso e ricevono i parametri ottimali. Interfaccia: web app o form interattivo.

C

Entrambi (uso misto)

Un backend Python riutilizzabile (libreria/API) con un frontend semplice per gli operatori e notebook per i ricercatori.

D

Demo / Proof-of-concept

Questa simulazione è solo per dimostrare il concept al team o ai partner. Bastano notebook ben strutturati con grafici chiari.
